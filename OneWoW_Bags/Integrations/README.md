# OneWoW Bags Integrations

This folder contains the integration implementation library.

## For Developers

**Looking to integrate your addon with OneWoW Bags?**

Go to: **[../API/](../API/)**

The API folder contains:
- Complete integration guide
- Code examples
- API reference
- Troubleshooting

## Files in This Folder

- **OneWoWBagsIntegration.lua** - Core integration library (automatically loaded by OneWoW Bags)
- **OneWoWBags_TransmogLootHelper.lua** - Example integration for TransmogLootHelper (can be copied by the addon)

## Quick Start

1. Read [../API/README.md](../API/README.md)
2. Copy an example from [../API/Examples/](../API/Examples/)
3. Customize for your addon
4. Add to your addon's `.toc` file
5. Done!
