# OneWoW - Data: Tradeskills

**Complete profession and recipe database for the OneWoW Catalog. Browse all recipes and crafting information across all professions.**

---

## What This Addon Does

This is a companion data addon for the OneWoW Catalog. It provides:

- **Recipe Database** - Complete list of recipes for all professions
- **Material Requirements** - What materials are needed for each recipe
- **Profession Information** - Skill requirements and progression
- **Crafting Costs** - Gold and material costs for crafting

---

## Required Addon

This addon requires:
- **OneWoW_Catalog** - The main Catalog addon that uses this data

---

## Installation

1. Extract the `OneWoW_CatalogData_Tradeskills` folder to your `World of Warcraft\_retail_\Interface\AddOns\` directory
2. Make sure `OneWoW_Catalog` is also installed
3. Restart World of Warcraft or type `/reload` in-game
4. Open the Catalog and go to the Tradeskills tab to see the data

---

## How to Use

1. Open the OneWoW Catalog
2. Click the "Tradeskills" tab
3. Select a profession from the list
4. Browse recipes or search for specific items
5. View material requirements, skill levels, and crafting details
