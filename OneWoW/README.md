# OneWoW - Central Hub

**The foundation for all OneWoW addons. A unified portal, smart tooltips, and everything you need to organize your World of Warcraft experience.**

---

## Features

### Portal Hub - Organized Teleportation
Quickly access all your portals, teleports, and hearthstones in one organized place.
- Browse all available portals and teleport items
- Add your favorites to quick access
- Filter by category: class portals, profession locations, instances, housing
- Automatic hearthstone detection
- Use portals and items with a single click
- Customize portal appearance and organization

### Item Status Indicators
See useful information at a glance when you hover over items in your bags:
- **Collections Status** - Know which mounts, pets, toys, and transmog items you already own
- **Item Level** - See gear item levels instantly
- **Junk Items** - Automatically marks low-value items for quick selling
- **Protected Items** - Highlights items you've marked as protected (won't be deleted accidentally)
- **Quest Items** - Easily spot quest-related gear
- **Crafting Materials** - Find reagents, recipes, and profession materials
- **Transmog Items** - Identify appearance sets and missing pieces
- **Warbound & Soulbound** - Know which items are bound to your character

### Collection Notifications
Get toast notifications (pop-up alerts) whenever you earn something new:
- New mounts discovered
- New pets collected
- New toys unlocked
- New recipes learned
- New transmog appearances obtained

### Enhanced Item Tooltips
Hover over any item to see extra helpful information:
- **Tooltip Enhancements** can hide Blizzard's default vendor sell-price line (on by default), which avoids the stock MoneyFrame tooltip path and related display issues; enable **Tooltips → Value → Show Vendor Price** if you want sell value only in the OneWoW tooltip section
- Whether you've collected this mount, pet, or toy already
- Custom notes you've added to items
- Gear upgrade suggestions
- What you're tracking and where to find it
- Recipe learning status (if you can learn it or already know it)
- Item categories and types
- For developers: Technical item/spell IDs when needed

### Universal Search
Quickly search through all your items, portals, and data without opening menus.

## Installation

1. Extract the `OneWoW` folder to your `World of Warcraft\_retail_\Interface\AddOns\` directory
2. Restart World of Warcraft or type `/reload` in-game
3. Type `/ow`, `/one`, `/onewow`, or `/1w` to open the addon

## Slash Commands

- `/ow` - Open OneWoW
- `/one` - Open OneWoW
- `/onewow` - Open OneWoW
- `/1w` - Open OneWoW
