# OneWoW - Data: Journal

**Complete instance and encounter database for the OneWoW Catalog. Provides detailed information about all dungeons and raids from every expansion.**

---

## What This Addon Does

This is a companion data addon for the OneWoW Catalog. It provides:

- **Instance Information** - Detailed data about all dungeons and raids
- **Encounter Details** - Information about bosses and encounters
- **Loot Tables** - What drops from each instance
- **Expansion Coverage** - All expansions from Classic through Midnight

---

## Required Addon

This addon requires:
- **OneWoW_Catalog** - The main Catalog addon that uses this data

---

## Installation

1. Extract the `OneWoW_CatalogData_Journal` folder to your `World of Warcraft\_retail_\Interface\AddOns\` directory
2. Make sure `OneWoW_Catalog` is also installed
3. Restart World of Warcraft or type `/reload` in-game
4. Open the Catalog and go to the Journal tab to see the data

---

## How to Use

1. Open the OneWoW Catalog
2. Click the "Journal" tab
3. Browse instances by expansion or search for specific raids/dungeons
4. Click on an instance to see detailed encounter information
5. View loot and other relevant data
