# OneWoW - Data: Vendors

**Complete vendor and item database for the OneWoW Catalog. Find vendors, what they sell, and where to get items.**

---

## What This Addon Does

This is a companion data addon for the OneWoW Catalog. It provides:

- **Vendor Locations** - Where to find every NPC vendor
- **Item Prices** - Cost for items sold by vendors
- **Vendor Inventory** - What each vendor sells
- **Currency Information** - What currencies vendors accept

---

## Required Addon

This addon requires:
- **OneWoW_Catalog** - The main Catalog addon that uses this data

---

## Installation

1. Extract the `OneWoW_CatalogData_Vendors` folder to your `World of Warcraft\_retail_\Interface\AddOns\` directory
2. Make sure `OneWoW_Catalog` is also installed
3. Restart World of Warcraft or type `/reload` in-game
4. Open the Catalog and go to the Vendors tab to see the data

---

## How to Use

1. Open the OneWoW Catalog
2. Click the "Vendors" tab to browse all vendors
3. Or use the "Item Search" tab to find where specific items are sold
4. View vendor location, inventory, and pricing information
