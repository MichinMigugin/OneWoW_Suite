# OneWoW - Shopping List

**A comprehensive shopping and crafting list manager. Build custom lists to track what you need to buy, craft, or farm.**

---

## Features

### Shopping List Management
- Create multiple shopping lists (Main List, shopping runs, crafting projects, etc.)
- Add items by name, ID, or drag-and-drop from bags
- Set quantities needed for each item
- Track what you already have vs. what you still need
- Mark items as complete when you've gathered them
- Organize with custom categories

### Multiple Lists
- Create parent lists and sub-lists (e.g., "Main Shopping" with sub-lists for different vendors)
- Switch between lists easily
- Set a default list to open on login
- Perfect for organizing different types of shopping (dailies, raids, farming)

### Crafting Integration
- Search your professions for recipes
- Automatically calculate materials needed for crafts
- See which items you need to complete recipes
- Integration with Catalog (if installed) to find crafting materials and recipes

### Bag Integration
- See which items you already have in your bags
- Automatic highlighting when you're holding the item
- Quick overlay badges showing progress on list items
- Search items across all your bags

### Alerts & Notifications
- Get alerts when you find items from your list
- Visual and audio notifications
- Disable alerts per list if needed

### Smart Features
- Search across all lists at once
- Sort items alphabetically or by priority
- Cross-character support (search your alts' bags too)
- Integration with Shopping List tooltips showing what you still need

### Customization
- Multiple color themes
- Multiple language support (English, Spanish, Korean, Russian, French, German)
- Customizable layout and sizing
- Quick-access minimap button

## Installation

1. Extract the `OneWoW_ShoppingList` folder to your `World of Warcraft\_retail_\Interface\AddOns\` directory
2. Extract the `OneWoW` folder (required dependency) to the same directory
3. Restart World of Warcraft or type `/reload` in-game
4. Type `/sl` or `/shoppinglist` to open the addon
